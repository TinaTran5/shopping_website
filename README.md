# Vegetarian Goods Shop
A modern, responsive e-commerce website built with Vue 3 to showcase and sell homemade vegetarian goods

### Built With
- Vue 3
- Vue Router
- SCSS / Nested CSS
- Composition API
- Local JSON Data Source


### Future Improvements
- Firebase Firestore product database
- Checkout + payment integration
- User login + favorites
- Mobile “slide-in” cart drawer
- Product filter/sort by type
